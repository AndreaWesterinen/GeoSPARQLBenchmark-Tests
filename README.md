# GeoSPARQL Compliance Benchmark Tests

This repository holds test details forked from the _GeoSPARQL Compliance Benchmark_ at https://github.com/OpenLinkSoftware/GeoSPARQLBenchmark. The original repository holds far more tests and is run using the HOBBIT Platform. This subset is used with the [modified Tests for Triplestore (TFT) platform](https://github.com/AndreaWesterinen/TFT).

The benchmark tests in this repository evaluate the GeoSPARQL compliance of RDF storage systems related to the Core, basic geometry and SimpleFeatures aspects of the [OGC GeoSPARQL standard](https://www.ogc.org/standards/geosparql). They cover requirements #1-4, 7-14, 19 and 22 of the standard, and represent the capabilities relevant to Wikidata.

For reference, the data, queries and results from the src/main/resources directory of the original benchmark are reorganized in the geosparql directory below.
